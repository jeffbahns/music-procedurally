Information
This program uses the seed number to generate rules on how each cell will either live, die, or come alive

The seeded numbers are anywhere between 0 and 2^512 (~154 digit #), randomly generated each time the game is restarted

Each row represents a different note (frequency) of the scale that is built at the beginning. You can select a scale and root note, or let them be randomized

Each column represents a time step, at which every row that is alive will be played as a note

(If a seed dies out, click cells to force back to life)
